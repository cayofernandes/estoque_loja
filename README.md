# estoque_loja
 Demanda gerada por ia para consolidar conceitos de programação

 Neste projeto, busquei solucionar o problema de gerenciamento de estoque de pequenos negócios. 
 Usei apenas programação procedurtal, mas pretendo orienta-lo a objetos mais tarde. 
 Treinei manipulação de arrays, controle de fluxo com o "do/while" e criação de funções com dados por referencia e com retorno de dados.
 Volterei para refatora-lo e otimizalo a web.
